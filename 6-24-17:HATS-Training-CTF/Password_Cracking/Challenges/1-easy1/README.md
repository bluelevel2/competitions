# easy1
First you need to do some online training! Hash the words.

## Questions
1. **MD5: hatsdabest**
2. **SHA1: P@ssw0rd**
3. **MD4: kendrickllama**
4. **SHA256: test**
5. **Whirlpool: letmein**

## Steps

### Question 1
1. `md5sum hatsdabest`

### Question 2
1. `sha1sum P@ssw0rd`

### Question 3
1. `[md4](https://linux.die.net/man/1/md4) kendrickllama`

### Question 4
1. `echo -n test | sha256sum`

### Question 5
1. 'Whirlpool Online Hash Generator'
2. Input 'letmein'
